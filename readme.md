# Similar to create-react-app

### Includes and installs:

-   React and React Dom
-   React Router Dom
-   Webpack
-   Babel
-   Eslint
-   Airbnb
-   Prettier
-   Husky
-   Material UI
-   Basic folder structure
-   Context
-   Custom hooks
-   Generic components (layout and nav)
-   404 handling
-   SCSS
-   Material themeing

## Usage

Run in terminal

    npx create-boris-app <YOUR APP NAME>

    cd <YOUR APP NAME>

    npm start
